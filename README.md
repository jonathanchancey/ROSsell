# ROSsell
CSE-180 -- Robotics Spring 2019

#### useful commands 

launch world with 

```
roslaunch husky_gazebo huskyfinal.launch
```

#### launch nodes with

```
rosrun finalproj exploreNode
rosrun finalproj obRec
rosrun rviz rviz
```


#### Print 1 message from topic
```
rostopic echo -n1 /my_cloud3
```
